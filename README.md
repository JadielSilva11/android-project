# Android Project

Este projeto é um site desenvolvido totalmente com HTML5 e CSS3, criado durante meu processo de aprendizado no curso de HTML5 e CSS3 do Curso em Vídeo.

O objetivo praticar conceitos de estruturação e estilização básicas de páginas web, desenvolvendo um site informativo sobre a história do mascote do Android.

🔗 Acesse o projeto publicado:
  https://SEU-USUARIO.github.io/android-project/

# Sobre o Projeto

O site apresenta a história do mascote do Android (Bugdroid), abordando:

  ✅ Seu design conceitual
 
  ✅ Processo criativo
 
  ✅ Evoluções ao longo das versões
 
  ✅ Seus criadores
   
  ✅ Consolidação da versão oficial

O conteúdo foi estruturado utilizando boas práticas de semântica e organização visual.

# Conceitos Aplicados

  - Estruturação semântica

  - Hierarquia de títulos

  - Tipografia personalizada

  - Paleta de cores temática

  - Responsividade básica

  - Incorporação de vídeos

  - Organização de conteúdo com listas

  - Estilização avançada com CSS

# Tecnologias Utilizadas

 - HTML5

 - CSS3
